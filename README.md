# Code management
### To clone the repo on your computer
```bash
git clone https://github.com/Locke789/Feel-trial
```
### To add modified files to a commit
```bash
git add theFile
```
### To generate and push a commit
```bash
git commit -m "a Clear message"
git push
```
### To get the modifications
```bash
git pull
```
